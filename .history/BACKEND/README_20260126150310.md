# Backend Service

## Overview

This repository contains the backend service responsible for APIs, authentication, database operations, and core business logic.

---


## Tech Stack

* Node.js
* Express.js
* MongoDB (Mongoose)
* Redis (Caching)
* JWT Authentication




---

## Setup

```bash
git clone <repo-url>
cd backend
npm install
```

Create `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
REDIS_URL=your_redis_url
```

Run server:

```bash
npm run dev
```
